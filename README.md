Welcome to the official repository for the **Machine Learning for Quantitative Investing Academy**. 

I developed and taught this curriculum for the **Warwick Business School Society** at the University of Warwick. The goal of this academy was to bridge the gap between theoretical statistical concepts and practical, programmatic deployment in financial markets, teaching students how to build data-driven investment strategies from scratch.

## 🎯 Course Overview

This repository contains the interactive Google Colab notebooks used during the live sessions. The curriculum is designed to take students from foundational statistical learning to advanced deep learning and portfolio optimization techniques, with a strict focus on financial applications.

### 📚 Curriculum Breakdown

* **Lesson 1: Introduction to ML in Finance (Regression & Classification)**
    * **File:** `Lesson_1_Intro_to_ML.ipynb`
    * **Topics Covered:** The role of supervised learning in quantitative finance. Building baseline models using Linear and Logistic Regression for stock price prediction and market direction classification.
    * **Financial Application:** Enhancing risk management by predicting conditional volatility using historical price movements and macroeconomic factors.

* **Lesson 2: Deep Learning and NLP for Market Sentiment**
    * **File:** `Lesson_2_Intro_to_NNs_and_NLP.ipynb`
    * **Topics Covered:** The mathematics behind artificial neurons, activation functions, and feedforward neural networks. Introduction to Natural Language Processing (NLP).
    * **Financial Application:** Extracting entities, analyzing sentiment, and building event-driven trading signals from unstructured financial text (e.g., earnings calls, news headlines).

* **Lesson 3: Time Series Analysis & Portfolio Optimization**
    * **File:** `Lesson_3_ML_for_Time_Series_and_Portfolio_Optimization.ipynb`
    * **Topics Covered:** Handling time-series data, predicting future asset returns, and evaluating model performance (MSE, Sharpe Ratio).
    * **Financial Application:** Modern Portfolio Theory (MPT). Using machine learning outputs to drive Mean-Variance Optimization, balancing expected returns against portfolio risk.

## 🚀 How to Use This Repository

The lessons are built as interactive Jupyter Notebooks and were originally executed in Google Colab to ensure all students had a uniform environment.

1. Clone the repository:
   ```bash
   git clone [https://github.com/ignaciovgl/ml-quant-investing-academy.git](https://github.com/ignaciovgl/ml-quant-investing-academy.git)
2. Open the notebooks locally using Jupyter, or upload them directly to Google Colab.

3. Install the required dependencies (if running locally):

```bash
pip install pandas numpy scikit-learn matplotlib pytorch
```
